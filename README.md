# Notes App

Full Stack Notes Application with Node.js and Express.

## Live Demo
[https://full-stack-notes-application-using-node-p7ar.onrender.com/](https://full-stack-notes-application-using-node-p7ar.onrender.com/)

## Features
- Create, read, update, delete notes
- Filter notes by category (Personal, Work, Study, Ideas)
- Persistent storage with JSON file
- Responsive design with Bootstrap 5.3
- Real-time updates with fetch API

## Technologies
- **Backend:** Node.js, Express
- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** Bootstrap 5.3, Bootstrap Icons
- **Database:** JSON file (lightweight storage)
- **Deployment:** Render.com

npm install
